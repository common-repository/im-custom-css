Tremendously easy-to-use custom CSS plugin for WordPress.

== Description ==

Need to add custom CSS to your WordPress website? IM Custom CSS will help you with just that. Tremendously easy-to-use custom CSS plugin for WordPress.

== Installation ==

How to install IM Custom CSS.

1. Upload the plugin directory 'im-custom-css' to the '/wp-content/plugins/' directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Use the Settings->IM Custom CSS screen to configure the plugin.

== Frequently Asked Questions ==

= My CSS code is not showing up, why? =

This could be for a lot of various reasons, ranging from wrongly written CSS code to being overwritten by some other CSS rules. We firstly recommend you to inspect your CSS code with your browser's "Inspector" which all the major browsers has today. If the problem still remains, we suggest seeking for help online.
